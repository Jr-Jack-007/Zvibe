# Firebase Setup

The app is wired to use Firebase Authentication, but `firebase-config.js` contains placeholder config values.

Replace these fields with the exact values from your Firebase console:
- `apiKey`
- `authDomain`
- `projectId`
- `storageBucket`
- `messagingSenderId`
- `appId`

After updating the config, email/password signup and login will work through Firebase Auth.
